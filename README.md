# SK Architectural Designs

A modern architectural landing page for SK Architectural Designs, showcasing residential services, portfolio, and pricing.

## Features

- **Responsive Design**: Built with Next.js and Tailwind CSS.
- **Project Gallery**: Filtering and lightbox functionality.
- **Services & Pricing**: Detailed breakdown of service packages.
- **Contact Form**: Integrated contact footer.
- **Theme**: Custom color palette and typography.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide Icons](https://lucide.dev/)
